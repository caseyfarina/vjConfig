# Midi Fighter 64 for Unity

MIDI input bridge for the [DJ Tech Tools Midi Fighter 64](https://www.midifighter.com/) and [Akai MIDI Mix](https://www.akaipro.com/midimix) in Unity 6.

## Install

1. Add Keijiro's scoped registry (**Edit > Project Settings > Package Manager**):
   - Name: `Keijiro` · URL: `https://registry.npmjs.com` · Scope: `jp.keijiro`
2. Install **Minis** from **Window > Package Manager > My Registries**.
3. Add this package via **+ > Add package from git URL**:
   ```
   https://github.com/caseyfarina/midiFighterForUnity.git
   ```

See `CLAUDE.md` for full architecture, API reference, and extension guide.
